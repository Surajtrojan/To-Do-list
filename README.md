An interactive TO-DO list app made using JavaScript(optimized code using JQuery library) with functionalities using GUI like add, delete buttons etc, having functionalities like:

  •	Add button with animation
  
  •	Fade Effect on deletion.
  
  •	Add button to show/hide activity writing bar.


Technologies used:
  
  •	HTML, CSS, JavaScript, JQuery
